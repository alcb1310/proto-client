# Protobuf Client

On this repo we will be adding a client for the [proto-server](https://github.com/alcb1310/proto-server) project. We will fetch data from the `protobuf` routes and
deserialize them to represent them in this app. We will only add one endpoint here for the demo.

## Getting Started

To run this application:

```bash
pnpm install
pnpm dev
```
