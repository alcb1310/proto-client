# Protobuf Client

On this repo we will be adding a client for the [proto-server](https://github.com/alcb1310/proto-server) project. We will fetch data from the `protobuf` routes and
deserialize them to represent them in this app. We will only add one endpoint here for the demo.

## Requirements

In order to follow this tutorial you will need the following:

- [proto-server](https://github.com/alcb1310/proto-server)
- [Protobuf compiler](https://github.com/protocolbuffers/protobuf/releases)


## Installation

To run this application:

```bash
git clone --recures-submodules https://github.com/alcb1310/proto-client.git
cd proto-client
pnpm install
pnpm dev
```
